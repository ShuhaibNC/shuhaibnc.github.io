# Cybersecurity Certifications

### Blue Team

#### Beginner
- [CompTIA Security+](https://www.comptia.org/certifications/security) - Entry-level certification covering core security concepts, threats, and risk management  
- [GIAC Security Essentials (GSEC)](https://www.giac.org/certifications/security-essentials-gsec/) - Hands-on security fundamentals including access control, cryptography, and networking  

#### Intermediate
- [Certificate of Cloud Security Knowledge (CCSK)](https://cloudsecurityalliance.org/education/ccsk/) - Cloud security fundamentals and best practices  
- [GIAC Certified Incident Handler (GCIH)](https://www.giac.org/certifications/certified-incident-handler-gcih/) - Incident detection, response, and handling techniques  

#### Advanced
- [Certified Information Systems Security Professional (CISSP)](https://www.isc2.org/certifications/cissp) - Advanced security architecture, management, and operations  
- [GIAC Certified Forensic Examiner (GCFE)](https://www.giac.org/certifications/certified-forensic-examiner-gcfe/) - Digital forensics and incident investigation  
- [Certified Information Security Manager (CISM)](https://www.isaca.org/credentialing/cism) - Security management, governance, and risk  

### Red Team

#### Beginner
- [CompTIA PenTest+](https://www.comptia.org/certifications/pentest) - Entry-level penetration testing and vulnerability assessment  
- [Certified Ethical Hacker (CEH)](https://www.eccouncil.org/programs/certified-ethical-hacker-ceh/) - Ethical hacking techniques and tools  

#### Intermediate
- [Offensive Security Certified Professional (OSCP)](https://www.offsec.com/courses/pen-200/) - Hands-on penetration testing and exploitation  
- [GIAC Penetration Tester (GPEN)](https://www.giac.org/certifications/penetration-tester-gpen/) - Penetration testing methodologies and tools  
- [Certified Red Team Operator (CRTO)](https://training.zeropointsecurity.co.uk/courses/red-team-ops) - Red team operations focused on Active Directory attacks  

#### Advanced
- [GIAC Exploit Researcher and Advanced Penetration Tester (GXPN)](https://www.giac.org/certifications/exploit-researcher-advanced-penetration-tester-gxpn/) - Advanced exploitation and vulnerability research  
- [Certified Information Systems Security Professional (CISSP)](https://www.isc2.org/certifications/cissp) - Broad, advanced security expertise  
- [Practical Network Penetration Tester (PNPT)](https://certifications.tcm-sec.com/pnpt/) - Real-world pentesting and reporting skills  
- [GIAC Security Professional (GSP)](https://www.giac.org/certifications/security-professional-gsp/) - Advanced, hands-on cybersecurity certification combining multiple GIAC certs
