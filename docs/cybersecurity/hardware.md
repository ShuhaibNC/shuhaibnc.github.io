# Cybersecurity Hardwares

## Top Hacking Gadgets

- [Raspberry Pi](https://www.raspberrypi.org/) - Low-cost credit card-sized computer running Linux with GPIO pins for hardware experimentation and learning programming  

- [USB Rubber Ducky](https://hak5.org/products/usb-rubber-ducky) - USB device that acts as a keyboard to inject keystrokes, execute payloads, and automate attacks  

- [Flipper Zero](https://flipperzero.one/) - Multi-functional device for interacting with RFID, NFC, infrared, and other wireless signals  

- [LAN Turtle](https://hak5.org/products/lan-turtle) - Covert network penetration testing tool for data gathering and remote access  

- [WiFi Pineapple](https://hak5.org/products/wifi-pineapple) - Wi-Fi auditing tool for reconnaissance and man-in-the-middle attacks  

- [Alfa Network Cards](https://www.alfa.com.tw/) - High-power Wi-Fi adapters used for packet capture and wireless security testing  

- [Ubertooth One](https://greatscottgadgets.com/ubertoothone/) - Open-source USB device for monitoring and analyzing Bluetooth signals  

- [Proxmark3](https://proxmark.com/) - RFID and NFC research tool for reading, writing, sniffing, and emulating cards  

- [Bash Bunny](https://hak5.org/products/bash-bunny) - Multi-function USB attack device that emulates HID, storage, and network interfaces to deliver payloads  

- [O.MG Cable](https://shop.hak5.org/products/omg-cable) - Malicious USB cable capable of keystroke injection and remote command execution over WiFi  

- [HackRF One](https://greatscottgadgets.com/hackrf/one/) - Software-defined radio for capturing and transmitting radio signals across a wide frequency range  

- [Yard Stick One](https://greatscottgadgets.com/yardstickone/) - Sub-1 GHz wireless transceiver for analyzing, capturing, and replaying ISM band signals
