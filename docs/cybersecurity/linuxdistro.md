# Linux Distros for Security & Privacy

- ⭐ [Kali Linux](https://www.kali.org/) - Penetration testing, digital forensics, and reverse engineering platform
- ⭐ [Parrot OS](https://parrotsec.org/) - Security, privacy, and software development tools based on Debian  
- ⭐ [BlackArch](https://blackarch.org/) - Arch-based penetration testing and security research distribution  
- [BackBox](https://www.backbox.org/) - Penetration testing and security analysis based on Ubuntu  
- [Fedora Security Lab](https://labs.fedoraproject.org/security/) - Security auditing, forensics, and system rescue tools on Fedora  
- [Qubes OS](https://www.qubes-os.org/) - Xen-based virtualization for strong isolation and enhanced privacy  
- [SIFT Workstation](https://www.sans.org/tools/sift-workstation/) - Incident response, digital forensics, and malware analysis toolkit  
- [Whonix](https://www.whonix.org/) - Enhanced anonymity and privacy using a built-in Tor gateway  
- [Pentoo](https://www.pentoo.ch/) - Gentoo-based penetration testing and security auditing system  
- [Security Onion](https://securityonionsolutions.com/) - Threat hunting and enterprise security monitoring platform  
- [Tails](https://tails.net/) - Live OS focused on privacy and anonymity through Tor
